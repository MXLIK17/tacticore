# TactiCore - Project Vision

## What is TactiCore?

TactiCore is a football draft and simulation game where players build legendary squads from football history and test those teams against realistic competition simulations.

The core fantasy behind TactiCore is simple:

> If you could build a football team from any era of history, could that team become the greatest team ever created?

The player is not managing a football club.

The player is building a football team.

Every draft should create a unique football story.

---

# Core Philosophy

TactiCore is built around four principles:

## 1. Football First

Every decision should prioritize the feeling of football.

The application should feel like:

- A tactical board
- A football stadium
- A team-building experience

It should never feel like:

- A business dashboard
- A database application
- A statistics management tool

Statistics exist to support the football experience, not replace it.

---

## 2. Meaningful Decisions

Every draft choice should matter.

Players should constantly think:

- Should I select Thierry Henry now or wait for another striker?
- Is Messi 2012 better for my team than Messi 2022?
- Do I need a defensive midfielder or another attacking player?
- Can this squad compete against the greatest teams in history?

The goal is not simply selecting the highest-rated players.

The goal is building the best possible team.

---

## 3. Historical Football Authenticity

Every footballer represents a specific season or tournament.

Different versions of the same player are treated as different players.

Examples:

- Lionel Messi 2006
- Lionel Messi 2012
- Lionel Messi 2022

These are separate footballers with separate ratings.

A player's rating represents their ability during that specific period.

Historical context matters.

---

## 4. Replayability

Every draft should feel different.

Players should want to restart and try:

- Different formations
- Different strategies
- Different eras
- Different football philosophies

The game should encourage experimentation.

---

# The Core Gameplay Loop

Every TactiCore experience follows the same cycle:
Choose Competition -> Choose Formation -> Draft Players -> Build Starting XI -> Simulate Competition -> Analyze Results -> Create New Draft 


Every feature added to TactiCore should improve this loop.

If a feature does not improve:

- Drafting
- Squad building
- Competition simulation
- Replayability

then it should not be prioritized.

---

# What TactiCore Is Not

TactiCore is not intended to become a football management simulator.

The following systems are intentionally excluded from Version 1:

- Transfers
- Contracts
- Player wages
- Finances
- Training systems
- Player development
- Career modes
- Club ownership
- Long-term management

These systems distract from the core experience.

The focus remains:

Build the team.
Test the team.
Compare the team against history.

---

# Player Fantasy

The player's goal is to answer questions like:

## Greatest Squad Challenge

Can my drafted XI defeat:

- Brazil 1970?
- France 1998?
- Spain 2010?
- Germany 2014?
- Argentina 2022?

---

## Tactical Challenge

Can different football philosophies succeed?

Examples:

- Attacking 4-3-3
- Defensive 4-2-3-1
- Balanced 4-4-2
- Modern 3-5-2

---

## Historical Challenge

Can players from different generations work together?

Examples:

- Ronaldo Nazario with Messi
- Zidane with Xavi
- Maldini with Van Dijk

---

# Competition Philosophy

Competitions should feel believable.

The simulation should follow these principles:

## Strong teams should usually perform well.

A historically great team should have a higher chance of winning.

---

## Football should remain unpredictable.

Upsets should happen.

A weaker team should occasionally defeat a stronger team.

This creates memorable moments.

---

## Results should tell stories.

A match should not only produce:
Brazil 2-1 France 


It should create:

- Goals
- Assists
- Player performances
- Tournament narratives

---

# Visual Identity

TactiCore should feel like a modern football tactics experience.

The visual identity should be:

- Minimal
- Modern
- Elegant
- Football-focused

The game should not imitate:

- EA FC Ultimate Team
- Football Manager
- Card collecting games
- Fantasy football websites

Instead, it should feel like:

> A professional tactical board where football history comes alive.

---

# Design Priorities

When making design decisions:

Prioritize:

1. Football clarity
2. User experience
3. Simplicity
4. Replayability
5. Visual polish

Avoid:

- Unnecessary complexity
- Excessive animations
- Feature overload
- Systems that distract from football

---

# Future Vision

Future versions of TactiCore may expand into:

- More competitions
- More historical teams
- Advanced chemistry
- Tactical adjustments
- More detailed statistics
- Additional game modes

However, expansion should always respect the original philosophy:

> Build the greatest football teams in history and discover whether they can truly compete.

---

# Final Principle

Whenever a development decision is unclear, ask:

> Does this make building and testing football teams more enjoyable?

If yes, continue.

If no, reconsider.
